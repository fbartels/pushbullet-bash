Welcome to pushbullet-bash

pushbullet-bash is a bash interface to the Pushbullet API. It can list your configured devices, contacts and channels and push different types of data to them.

Set your Pushbullet Access Token by calling pushbullet-bash with the argument setup (e.g. pushbullet setup) or by creating the file $HOME/.config/pushbullet and adding the line PB_API_KEY= to it.

More information can be obtained from the project page or the included help.
